# Daily Projects Portfolio

This repository contains a collection of small projects I build regularly to improve my software engineering skills and document my learning progress.

## About

I created this repository to practice building software consistently and to maintain a portfolio of hands-on projects that demonstrate my problem-solving approach and technical growth.

Each directory represents a standalone mini project exploring a specific concept, tool, or technology.

## Goals

* Build projects consistently to strengthen programming fundamentals
* Experiment with different languages, frameworks, and tools
* Document learning through practical implementation
* Maintain a portfolio of work to showcase when applying for software engineering roles

## Structure

Each project is contained in its own folder:

```text
project-name/
  README.md
  source files
```
